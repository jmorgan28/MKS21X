Clone - gives you a local copy of the git repository.

Add - adds file to the staging area.

Commit - commits the changes you have made. Add a comment with it.

Push - send committed changes back to git hub. Updates repository with local changes.

Pull - updates local with newest repository information.

show - shows one or more objects. 

checkout - restores it to how it was at last commit.

reset - allows you to rewind the project to a chosen commit

diff - displays what has been changed

status - tells you status of staging area compared to working directory

log -shows the commits you have done
